Write Native Enterprise Mobile Apps with Web Skills

Use your existing HTML, CSS, JavaScript and Ruby skills to build native apps for all popular operating systems: iOS, Android, Windows Phone, Windows Mobile and Windows 8. RhoElements is the only modern framework focused on the needs of enterprise apps.

The Only Enterprise Mobile Framework

RhoElements is not just an API façade. It assists in the full lifecycle and breadth of app development. It offers a built-in app generator to get you going quickly. Its MVC pattern lets you build maintainable apps that separate business logic from interface. ORM lets you easily build information-intensive databased oriented apps without writing SQL. All automatically synchronized for easy offline data usage, critical for enterprise apps.

Largest API Set of Any Framework

RhoElements offers dozens of built-in capabilities, far more than any of the consumer app-oriented frameworks. These include barcode reading, magnetric stripe card reading, Bluetooth, printing, NFC, RFID, battery life indication.

The Most Secure Mobile Platform

RhoElements performs automatic data encryption for data at rest security. It also offer public/private key(PKI) encryption. It offers an optional FIPS certified encryption algorithm. No other framework offers these features. Use of th RhoConnect data synchronization component automates integrating with an enterprise's Single Sign-on Infrastructure.

Open Source and Massive, Mature Community

RhoElements is built on the free open source Rhodes framework. It merely adds several enterprise relevant capabilities such as barcoding and advanced security. Rhodes is still the most powerful free framework, offering MVC and ORM and many other features not even conceived of by consumer frameworks such as PhoneGap. Rhodes was started in August 2008 as the first smartphone framework, and has maintained its lead in features over the last six years. Here are just some the thousands of Rhomobile apps.
